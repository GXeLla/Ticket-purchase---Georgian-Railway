# 🚆 Georgian Railway Ticket Purchase

A fully functional **train ticket purchase website** built with **HTML, CSS, and JavaScript**, using **real APIs** to fetch data and manage user authentication.
The project simulates a real train booking experience where users can search for routes, select travel details, and manage their account.

🔗 **Live Website:**
https://gxella.github.io/Ticket-purchase---Georgian-Railway/

---

## 📌 Features

* 🔎 **Search Trains** – Select departure and arrival cities from dynamically loaded stations.
* 📅 **Date Picker** – Choose travel dates with validation (past dates disabled).
* 🎟 **Ticket Counter** – Increase or decrease the number of tickets.
* 🔄 **Swap Cities** – Quickly switch departure and arrival locations.
* 👤 **User Authentication**

  * Register
  * Login
  * Logout
* 🧑 **User Profile**

  * Fetch user data from API
  * Edit profile information
  * Change password
  * Delete account
* 🖼 **Avatar Preview** – Preview profile picture before saving.
* 🧾 **Results Page** – Displays travel information using URL query parameters.
* ⚡ **Dynamic UI** – Dropdowns, modals, and interactive components powered by JavaScript.

## 🔗 APIs Used

### Railway API

Used to fetch available train stations.

```
https://railway.stepprojects.ge/api
```

### Everrest Educata API

Used for authentication and user management.

```
https://api.everrest.educata.dev
```

Functions implemented with the API:

* User registration
* Login / Logout
* Fetch user by ID
* Update user profile
* Change password
* Delete account
